# Aura Chatbot

Welcome to the Aura Chatbot project! Aura stands for **Automated User Response Assistant**. It is a simple and interactive chatbot built using HTML, CSS, and Python-Flask. This project demonstrates how to create a web-based chatbot interface that can send and receive messages.

## Features

- **Interactive Chat Interface**: Users can type messages and receive responses in real-time.
- **Responsive Design**: The chatbot interface is designed to be responsive and works well on various screen sizes.
- **Smooth Animations**: Messages fade in and have hover effects for better user experience.
- **Customizable**: You can easily change the bot's name and customize the appearance.

## Technologies Used

- HTML
- CSS
- Python-Flask
- Fetch API for server communication
